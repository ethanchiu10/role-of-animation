How do I add animations to my site?

Techniques

  • CSS Transitions
  • CSS Animations
  • JavaScript

CSS Transitions

ball background: red transition: all 1s ease 0s ball.active background: blue transform: translateY(200px)

CSS Transitions - When to use

  • Simple state-based animations
  • Can smoothly cancel animation

CSS Animations (Keyframes)

@keyframes pulse 0% transform: scale(1) 100% transform: scale(0.6) circle animation: pulse 0.5s infinite alternate

CSS Animations - When to use

  • Complex multi-step animations
  • Looping animation
  • Animation that can run in either direction

JavaScript

  • element.animate()
  • GSAP, TweenJS, Velocity.js
  • Box2D, React Motion

JavaScript - When to use

  • Dynamic animations
  • Physics
  • Scroll / Touch / Drag
chenglou.me/react-motion

Performance

  • Transform
  • Opacity
  • (Some) CSS Filters

Future

  • Navigation Transitions
  • New CSS Properties that can be transitioned without causing reflows
